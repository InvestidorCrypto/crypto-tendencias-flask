# 📈 Cripto Tendências

Este é um site simples feito com Flask, que permite publicar tendências e pré-vendas de criptomoedas com imagem e descrição.

## 🚀 Funcionalidades
- Página inicial com lista de posts
- Cadastro de novos posts com upload de imagem
- Armazenamento local com JSON

## 🛠️ Requisitos
- Python 3.8+

```bash
pip install -r requirements.txt
```

## ▶️ Como rodar localmente
```bash
python app.py
```

Acesse: [http://localhost:5000](http://localhost:5000)

## ☁️ Deploy no Render
1. Crie um repositório no GitHub com os arquivos do projeto.
2. Vá para [https://render.com](https://render.com) e conecte sua conta do GitHub.
3. Crie um novo Web Service:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python app.py`
   - Runtime: Python 3.x
4. Pronto! Seu site estará online.
